# Podcastify Frontend

A beautiful React application that turns any book into a podcast.

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Features

- Elegant homepage with podcast studio aesthetic
- Book search functionality
- Upload your own books
- Beautiful typography using Playfair Display font

## Tech Stack

- React 18
- Vite
- Lucide React (icons)
